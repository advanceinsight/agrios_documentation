Welcome to the official AgriOS documentation
============================================

AgriOS is an open-source platform designed to help farms and agribusinesses manage operations across production, inventory, sales, purchasing, accounting, and more. 

Whether you are a farmer looking to streamline daily operations, an integrator deploying AgriOS for customers, or a developer extending the system, this documentation is organized to get you productive quickly.

Who this documentation is for
============================================

- Farmers and business users: step-by-step guides for common tasks and workflows.
- System administrators: deployment, hosting and database management guidance.
- Developers and module authors: design overview, development practices, and integration guidance.
- Partners and contributors: information about partnerships, distribution, and contribution processes.
